# Boltcall SEMrush Easy-Win Playbook

SEMrush should be used here as a decision engine for buyer pages, not a reporting dashboard.

The operating goal is simple:

1. find low-difficulty, high-intent vertical phrases
2. map each winning phrase to one clear page
3. improve existing money pages before creating net-new pages
4. track only pages that can produce booked calls

## Primary Workflow

Run SEMrush in this order:

1. Keyword Magic Tool
2. Keyword Gap
3. Organic Research
4. Position Tracking
5. Site Audit and On Page SEO Checker

Use the tools to decide which page should exist next, not to create a giant keyword spreadsheet that never becomes a page.

## Acceptance Rules

A phrase is worth building only if it passes all of these:

- clear buying or solution intent
- specific vertical or specific operational problem
- realistic difficulty for Boltcall's current authority
- maps to one page with one CTA
- can be answered directly in the first 100 to 150 words

Default bias:

- prefer lower difficulty over higher volume
- prefer vertical noun plus solution noun
- prefer "best", "for", "vs", "alternative", "answering service", "missed call", and "speed to lead"
- deprioritize broad research queries unless they clearly feed a money page

## Page Type Rules

- `/industries/*` for direct trade-intent buyer pages
- `/blog/*` for question-driven but still commercial buyer searches
- `/compare/*` or `/comparisons/*` for evaluation phrases
- tools or calculators only when repeated problem-intent patterns show up in SEMrush

The current Week 1 phrase stack is:

1. `roofer answering service`
2. `ai receptionist for hvac`
3. `ai receptionist for roofers`
4. `best ai receptionist for home services`
5. `how fast should a contractor respond to a new lead`

Execution rules:

- improve an existing page first if one already targets the phrase
- create one page only when there is no clean existing destination
- every important page should gain contextual internal links from related assets

## Daily Loop

Spend 45 to 60 minutes on one focused pass:

1. check one SEMrush snapshot
2. review one competitor gap
3. inspect one candidate cluster
4. choose one highest-leverage action
5. log the phrase, page, reason, and next target

High-leverage actions should usually be:

- stronger first 150 words
- FAQ or schema tightening
- new internal links
- one new page only when page improvement is weaker

## Weekly Loop

Once per week:

1. refresh Position Tracking
2. run Site Audit
3. review On Page SEO Checker for money pages only
4. prune weak phrases from the backlog
5. set the next five-page list

## Success Checks

The operating standard is:

- one phrase maps to one page
- the answer appears immediately near the top
- the page has relevant internal links
- the FAQ and schema match the search intent
- the tracking list stays tight
- each week ships either one new money page or two to three meaningful upgrades
