# Zapier Marketplace Submission Notes

## Listing

**App name:** Boltcall

**Short description:** Instantly call, text, and book new local-service leads from any Zapier workflow.

**Long description:** Boltcall is an AI speed-to-lead system for local service businesses. Use the Zapier integration to send new leads from forms, ads, spreadsheets, CRMs, and booking tools into Boltcall. Boltcall captures the lead, starts the first-touch response, and keeps the lead available for downstream automations.

## Test account checklist

- Boltcall workspace with one active API key.
- At least one test lead already present so `New Lead` returns sample data.
- A phone number and speed-to-lead agent configured if reviewers need to see the instant call path.

## Screenshots checklist

- API key setup in Boltcall.
- Zapier auth screen.

- Successful test result with `first_touch_status`.
