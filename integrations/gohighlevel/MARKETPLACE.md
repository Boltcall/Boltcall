# GoHighLevel Marketplace Submission Notes

## Listing

**App name:** Boltcall

**Short description:** AI speed-to-lead calling and SMS follow-up for HighLevel agencies.

**Description:** Boltcall connects HighLevel leads to instant AI phone and SMS follow-up. Agencies can push new contacts or opportunities into Boltcall, start first-touch response automatically, and keep HighLevel updated with lead status and source tags.

## Review checklist

- OAuth app credentials.
- Install and uninstall webhook handling.
- Contact create/update demo.
- Example workflow: New HighLevel contact -> Boltcall speed-to-lead.
- Support docs for Location ID, scopes, and contact permissions.
